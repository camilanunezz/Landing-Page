Project Description

Database for University Bus Routes*

To start, first we configured a JSON file containing an API that contains university bus route information, next this information was parsed. The next part of our program handled data logic and queries written in SQL. A JDBC Connection was used to write queries that: Create your Tables, Insert your data into the Tables, Query (Select) information from the Tables, Clear the tables. To organize the information, we used SQLite databases relevant to diffierent bus lines, stops, and route order. Lastly, another part of the program included implementing an algorithm that found the closest bus stop to a given location to complete the program.

*This project was created in a team in a Software Development course. The code to this project can not be uploaded due to Honor Code policies at the University of Virginia.
