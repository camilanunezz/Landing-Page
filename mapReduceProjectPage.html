Project Description

MapReduce Implementation in Simulated Distributed System*

This project used Go to acheive the following: Enhanced a MapReduce library by improving the implementation of parallel task execution across worker threads. Developed fault-tolerance mechanisms to manage worker failures, ensuring reliable task execution in simulated distributed system environment.

*This project was created in a Cloud Computing course and completed independently. The code to this project can not be uploaded due to Honor Code policies at the University of Virginia.
